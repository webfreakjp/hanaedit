# 残タスク

## 配布

- `.app` bundle の生成を安定化する。
- Developer ID 署名と notarize に対応する。
- Homebrew Cask の公開フローを整える。
- GitHub Releases にビルド済み成果物を添付する。
- バージョン更新、tag 作成、Homebrew Cask 更新をスクリプト化する。

## エディタ基本機能

- 設定保存に対応する。
- 最近使ったファイルを管理する。
- フォント、文字サイズ、タブ幅、折り返しを設定できるようにする。
- 文字コードと改行コードを保存時に選択できる UI を追加する。
- 未保存変更がある複数ウィンドウの終了確認を改善する。

## 検索・置換・grep

- grep 結果をクリックして該当ファイルと行へジャンプする。
- grep の対象ファイルパターンと除外パターンを保存する。
- grep の結果一覧をテーブル表示にする。
- 検索/置換パネルの履歴を保存する。
- 正規表現エラーの表示を分かりやすくする。

## 矩形編集

- 日本語 IME 入力時の矩形編集をさらに検証する。
- タブ文字を含む行での矩形位置計算を改善する。
- 矩形選択中のコピー/カット/ペーストの挙動を整理する。
- 行末より右側への矩形入力時の空白補完を設定可能にする。

## 表示

- 不可視文字の表示記号をカスタマイズできるようにする。
- 行番号、現在行、検索結果のハイライト表示を改善する。
- ダークモードでの配色を調整する。
- タイプ別カラー設定、キーワード強調、アウトライン解析を検討する。

## 互換性・品質

- 文字コード round trip のテストを追加する。
- 改行コード維持のテストを追加する。
- 検索/置換/grep の edge case テストを追加する。
- 矩形編集のユニットテストを追加する。
- 大きなファイルを開いたときの性能を確認する。
